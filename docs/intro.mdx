# Introdução

Bem-vindo à documentação das APIs da Notax. Aqui você encontra tudo que precisa para integrar nossas soluções na sua aplicação.

## APIs disponíveis

| API | Versão | Descrição |
|-----|--------|-----------|
| [Notax Pay](/notax-pay) | v1 | Pagamentos, contas bancárias e convênios |

## Autenticação

Todas as APIs utilizam autenticação via **Bearer Token**. Inclua o token no header de todas as requisições:

```http
Authorization: Bearer {seu-token}
```

## Base URL

| Ambiente | URL |
|----------|-----|
| Produção | `https://api.notax.com.br` |

## Formato das respostas

Todas as respostas são retornadas em formato JSON. Erros seguem o padrão [RFC 7807 (Problem Details)](https://datatracker.ietf.org/doc/html/rfc7807):

```json
{
  "type": "https://tools.ietf.org/html/rfc7231#section-6.5.1",
  "title": "Bad Request",
  "status": 400,
  "detail": "Descrição do erro"
}
```

## Suporte

Em caso de dúvidas, entre em contato através do site [notax.com.br](https://notax.com.br).
